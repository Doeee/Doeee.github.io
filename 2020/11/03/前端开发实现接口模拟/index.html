前端开发实现接口模拟

前端开发接口模拟的两种方式

一、mockapi

1、创建数据的api

https://www.mockapi.io/login

登录这个网站，可以用git账户直接登录

点击新建项目可以设置json格式，创建好后进度条可添加数据，如下进度条是10就是10条数据

获取数据输入api

2、测试数据是否可以正常的得到数据

①可以用postman发送get指令可以拿到创建的数据

用postman向接口发送post请求

返回mockapi可以看见新添加的数据

②也可以通过页面F12在控制台获取api的数据

3、在代码中拿到api中的数据

①通过jquery

在代码中拿到api 数据并打印出来，首先要在项目中导入jquery

首先头部引入

import $ from 'jquery';

也可以如下按需导入jquery因为只需要getjson的方法

然后控制台安装jquery

yarn add jquery

然后重启服务

npm start

然后拿到数据并且打印出来如下

②还可以通过axios库

首先在控制台安装axios

yarn add axios

在js文件头部导入这个库

import axios from 'axios'

然后用axios,get方法获取数据（注意axios处理错误和jquery不同，需要用catch接受数据，错误信息不是errprresponseText而是error.message）

（https://github.com/axios/axios#handling-errors

这个网址可看axios的文档）

二、使用json server

1、全局安装

npm install -g json-server

查看版本号

json-server -v

2、开启监听端口

创造json数据，在任意文件夹下（我用的是项目文件夹）

json-server --watch db.json --port3004

默认端口的3000，可以通过”–port 端口号“来修改监听端口

这个项目文件夹下则会新生成一个db.json文件

关闭监听端口

然后把mockapi中的数据复制进来

4、再次启动监听端口

5、用postma发送指令

可以在postman中get到资源数据

按照json的格式去创建数据，并且通过api添加数据点击send
